# Backlog do Produto

Este backlog descreve as principais funcionalidades do **Sistema de Apoio a Intercambistas**.

## Histórias de Usuário

História 1 – Como estudante, quero me cadastrar no sistema para poder acessar as funcionalidades básicas.
História 2 – Como estudante, quero realizar login no sistema para acessar minha área restrita.
História 3 – Como estudante, quero visualizar uma lista de países disponíveis para intercâmbio.
História 4 – Como estudante, quero visualizar universidades associadas a cada país.
História 5 – Como estudante, quero visualizar um checklist de documentos necessários para o intercâmbio.
História 6 – Como administrador, quero cadastrar e editar países disponíveis para intercâmbio.
História 7 – Como administrador, quero cadastrar universidades vinculadas a um país.
História 8 – Como administrador, quero cadastrar a lista de documentos exigidos para o intercâmbio.
