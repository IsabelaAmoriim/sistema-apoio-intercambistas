# Organização dos Sprints

O desenvolvimento do projeto será dividido em três sprints.

## Sprint 1 – Ajustes Iniciais
- Criação do repositório no GitHub
- Definição das tecnologias
- Definição do escopo do sistema
- Escrita e organização das histórias no backlog

## Sprint 2 – Funcionalidades Básicas
- Cadastro de estudantes
- Login de estudantes
- Estrutura inicial do sistema

## Sprint 3 – Funcionalidades Complementares
- Visualização de países e universidades
- Acompanhamento de requisitos e status
- Funcionalidades administrativas simples
